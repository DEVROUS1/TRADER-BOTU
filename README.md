# KriptoGraf Finans Trader Botu

Bu bot, MEXC borsasını takip ederek teknik analiz yapar, ML destekli fiyat tahminleri oluşturur ve Telegram üzerinden bildirim gönderir.

## 🚀 Hugging Face Kurulumu

1. Yeni bir **Space** oluşturun (SDK: Streamlit seçin).
2. Tüm dosyaları bu repoya yükleyin.
3. **Settings > Variables and Secrets** kısmına aşağıdaki anahtarları ekleyin:
   - `TELEGRAM_TOKEN`
   - `TELEGRAM_CHAT_ID`
   - `GEMINI_API_KEY`

Botunuz 7/24 çalışmaya başlayacaktır!
